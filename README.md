# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview
This project is a Machine Learning-based system that predicts house prices using regression models. It estimates property value based on features such as area, bedrooms, bathrooms, floors, parking, and age of the house.

It includes data analysis, visualization, model training, and prediction using ML techniques.

---

## 🎯 Objective
To build an end-to-end regression system that:
- Predicts house prices accurately
- Analyzes important features affecting price
- Compares multiple ML models
- Provides insights for real estate decision making

---

## 🧠 Problem Statement
Real estate pricing is often subjective and inconsistent. This project solves this problem by using machine learning to predict fair property prices based on historical data and property features.
